Getting Started

  1. Create an account.
  2. Add trusted emergency contacts.
  3. Enable GPS location services.
  4. Customize emergency settings.
  5. Use the SOS button when assistance is needed.

Frequently Asked Questions

Does SafeRoute track me constantly?

No. Location sharing occurs only when enabled or during emergencies.

Can I add multiple emergency contacts?

Yes. Users can add several trusted contacts.

Can I use SafeRoute while traveling?

Yes. The application is designed for use in multiple locations.
